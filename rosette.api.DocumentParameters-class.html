Class DocumentParameters

source code

           object --+    
                    |    
_DocumentParamSetBase --+
                        |
                       DocumentParameters

Parameter object for all operations requiring input other than translated_name. Two fields, content and inputUri, are set via the subscript operator, e.g., params["content"], or the convenience instance methods DocumentParameters.load_document_file and DocumentParameters.load_document_string.

Using subscripts instead of instance variables facilitates diagnosis.

If the field contentUri is set to the URL of a web page (only protocols http, https, ftp, ftps are accepted), the server will fetch the content from that web page. In this case, content may not be set.
